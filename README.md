# Noted


## Getting Started
1. Install Ruby and Jekyll
   Refer to the [Docs](https://jekyllrb.com/docs/installation/)

2. Clone the repo
    ```sh
    git clone https://github.com/Kairos-T/noted
    ```
3. Install dependencies
    ```sh
   bundle
    ```
   
4. Run the Local Server
    ```sh
    bundle exec jekyll s
    ```
